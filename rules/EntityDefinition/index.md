# EntityDefinition
### A custom object Description is required
Field: `Description`


### A custom object Name must be PascalCase
regex: `^[A-Z][A-Za-z]*$`  
Field: `qualifiedApiName`
#### Example
InsurancePolicy  

### Less than 100 custom fields
Field: `qualifiedApiName`


### Maximum one custom trigger
Field: `qualifiedApiName`
