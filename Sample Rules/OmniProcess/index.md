# OmniProcess
## Omniscript Type should be pascal case
Field: `Type`   
Regex: `^[A-Z][A-Za-z0-9]*$`    



## Omniscript SubType must be PascalCase
Field: `SubType`   
Regex: `^[A-Z][A-Za-z0-9]*$`    



## Omniscripts must have a description
Field: `Description`   



## Omniscript must have less than 100 elements
Field: `OmniProcessElements.totalSize`
